# Design

This directory stores detailed design diagrams and reusable design definitions.

## Structure
- `10_Diagrams/` - detailed design diagrams composed from reusable definitions
- `20_Definitions/` - reusable design building blocks included by diagrams

## Usage
1. Put reusable classes/modules/hardware definitions in `20_Definitions`.
2. Compose detailed diagrams in `10_Diagrams` using `Include` rows.
3. Keep diagram files focused on relationships and behavior, not repeated declarations.
